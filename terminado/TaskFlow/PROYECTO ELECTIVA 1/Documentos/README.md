# TaskFlow — Proyecto académico

**TaskFlow** es una aplicación Android nativa para la **gestión de tareas personales**, desarrollada como proyecto de electiva con **Kotlin**, **Material Design 3**, arquitectura **MVVM** y persistencia con **Room**.
