# Diseño de interfaz de usuario

La aplicación tendrá la siguientes pantallas, Una de Inicio; Crear tarea y un Calendario. La interfaz de usuario de la aplicación Metodo Cosecha toma de referencias en la interfaz de usuario de aplicaicones similares

Los diseños completos esta disponible en [Figma](https://www.figma.com/proto/qvajwkveMSGbfNRHbFaq3S/MOCKUPS?node-id=0-1&t=AcKDt7AcjkSmyVem-1) . Los colores a usar estar[an basados en una combinación de Colores Secundarios. A continuación se presentan las principales pantallas:

1. Pantalla 1: INICIO
   Cuando el usuario ingrese a la aplicacion se encontrara con este organizador de tarea en listas con sus colores reconocibles

![screen1](Imagenes/image.png)


2. Pantalla 2: CREAR TAREA
   Pasando por una de las funciones de la app que esta intuitivamente con un boton de "+", sigue el de crear una tarea o pendiente con        varios aspectos para que se pueda separar de otras tareas

![screen2](Imagenes/Imagen2.png)

3. Pantalla 3: Calendario
   Otra de las funciones es Calendario donde se ven las tareas como tal mensuales clasificadas por importancia, fecha limite (si tiene),      aunque tambien se podran ver por semana o volver a las diarias
   
![screen2](Imagenes/Imagen3.png)
