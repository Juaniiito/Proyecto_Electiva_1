# Ideas de proyecto

## Idea 1 (Notif)


Una aplicación en la que se puedan añadir recordatorios y notas, con el fin de recordar algún evento importante además de añadir notas, además de colocar la fecha para cierto evento. Al momento de seleccionar una fecha sea guardado en la "base de datos" y una vez llegue la fecha la aplicación pueda notificar en estilo "alarma"

Aplicaciones similares:
-[Notification notes](https://play.google.com/store/apps/details?id=com.hasmetd.notificationnotes)
-Parecido a la aplicación de "Reloj" que tenemos en los celulares Android 


## Idea 2 (CampusConnect)

CampusConnect es una aplicación móvil tipo red social diseñada para entornos académicos. Permite a los usuarios compartir publicaciones, interactuar mediante likes o comentarios, gestionar perfiles básicos y comunicarse dentro de una comunidad estudiantil. La app busca mejorar la interacción entre estudiantes, facilitar el intercambio de información y fortalecer la comunicación universitaria.

Aplicaciones similares:

-[Facebook](https://play.google.com/store/search?q=facebook&c=apps)
-[Reddit](https://play.google.com/store/search?q=reddit&c=apps)
-[Instagram](https://play.google.com/store/search?q=instagram&c=apps)


## Idea 3 (TodoPulse)


Una aplicación móvil Android para la gestión de tareas personales, desarrollada como proyecto académico. La aplicación permite crear, editar, eliminar y organizar tareas de manera eficiente, clasificándolas en secciones inteligentes como Hoy, Próximos días y Algún día, facilitando la priorización y el seguimiento de actividades.

Aplicaciones similares:

-[Todoist](https://play.google.com/store/search?q=todoist&c=apps)
-[Microsoft To Do](https://play.google.com/store/search?q=to%20do&c=apps)
-[TickTick](https://play.google.com/store/search?q=TickTick&c=apps)
