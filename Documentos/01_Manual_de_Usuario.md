# TaskFlow — Manual de usuario

## 1. ¿Qué es TaskFlow?

TaskFlow le permite **crear, organizar, buscar y completar** tareas personales desde el teléfono. Las tareas se agrupan automáticamente en tres zonas de trabajo más una lista de completadas.

## 2. Pantalla principal

### 2.1 Barra superior

- Muestra el nombre **TaskFlow**.
- Debajo encontrará el **campo de búsqueda**: escriba palabras clave para filtrar por título, descripción o categoría.

### 2.2 Filtros por prioridad

Debajo de la búsqueda hay **chips** (etiquetas pulsables):

- **Todas:** quita el filtro de prioridad.
- **Alta, Media, Baja:** muestra solo tareas de esa prioridad (combinado con el texto de búsqueda si lo hay).

### 2.3 Listas inteligentes

Las tareas **no completadas** aparecen bajo encabezados:

| Sección | Significado |
|---------|-------------|
| **Hoy** | Prioridad alta (inmediata) o fecha de vencimiento hoy o anterior (incluye retrasadas). |
| **Próximos días** | Fecha de vencimiento en el futuro (posterior al día actual). |
| **Algún día** | Sin fecha definida (y que no entren en “Hoy” por prioridad alta). |

Las tareas **completadas** aparecen al final bajo **Completadas**.

### 2.4 Botón flotante (+)

Pulse el botón **redondo con +** para **crear una nueva tarea**. Se abrirá el formulario de edición.

### 2.5 Abrir o editar una tarea

Pulse **sobre la tarjeta** de una tarea (no solo en la casilla) para **editarla**.

### 2.6 Marcar como completada

Use la **casilla** a la izquierda del título. Las tareas completadas pasan a la sección inferior **Completadas** y dejan de programarse recordatorios por fecha.

### 2.7 Eliminar una tarea

**Deslice la tarjeta hacia la izquierda.** Aparecerá un cuadro de confirmación. Si confirma, la tarea se elimina de forma permanente.

---

## 3. Formulario de tarea (crear / editar)

### 3.1 Campos

| Campo | Descripción |
|-------|-------------|
| **Título** | Obligatorio. Nombre breve de la tarea. |
| **Descripción** | Texto libre con detalles. |
| **Prioridad** | Alta, media o baja (botones en fila). |
| **Fecha** | Opcional. Use **Elegir** para abrir el calendario; **Quitar** borra la fecha. |
| **Categoría** | Puede escribir o elegir sugerencias (Personal, Estudio, Trabajo, etc.). |

### 3.2 Guardar

Pulse el icono de **guardar** en la barra superior del formulario (o la opción de menú equivalente). Si el título está vacío, la aplicación mostrará un mensaje de error en el campo título.

### 3.3 Volver atrás

Use la **flecha atrás** de la barra superior para volver a la lista sin guardar (los cambios no guardados se perderán).

---

## 4. Recordatorios (notificaciones)

Si concede el permiso de **notificaciones** (Android 13 en adelante), TaskFlow puede mostrar un **recordatorio** el día de la fecha de la tarea (lógica programada en la aplicación; horario según implementación del proyecto).

Si deniega el permiso, la app seguirá funcionando, pero puede no mostrar recordatorios.

---

## 5. Modo claro y oscuro

TaskFlow respeta el **tema del sistema** (claro u oscuro) mediante Material 3 **DayNight**. Cambie el tema en los ajustes del dispositivo para ver la interfaz adaptada.

---

## 6. Consejos de productividad

1. Use **prioridad alta** solo para lo verdaderamente urgente; así la sección **Hoy** sigue siendo útil.  
2. Programe **fechas** para lo que tiene día concreto; deje sin fecha lo que es “cuando pueda”.  
3. Combine **búsqueda + filtro de prioridad** para encontrar rápido una tarea entre muchas.
